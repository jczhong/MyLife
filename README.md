## MyLife

a personal tasks managment tool

## Installation

    curl https://install.meteor.com/ | sh
    git clone https://github.com/jczhong/MyLife.git
    cd MyLife
    meteor run
    

## Custom markdown syntax

    //Create a task
    - <Item value='Do task1' close/>
        - <Item value='Do sub task1' />
    - <Item value='Do task2' />
